# Natan TopUp - Telegram Bot & Admin Dashboard

A complete Telegram bot for game top-up services with an admin dashboard.

## Features

- 🤖 Telegram Bot for ordering
- 📊 React Admin Dashboard
- 💳 Multiple payment methods
- 🎮 Support for PUBG, Free Fire, TikTok, Telegram Premium
- ⚡ Instant delivery via Ragner API
- 📦 Manual delivery option
- 📱 Mobile responsive dashboard

## Tech Stack

- **Backend**: Node.js, Express, PostgreSQL
- **Frontend**: React, Tailwind CSS
- **Bot**: Telegraf
- **Deployment**: Railway

## Setup Instructions

1. Clone repository
2. Copy `.env.example` to `.env` and fill your values
3. Run `npm install`
4. Run `cd admin-dashboard && npm install`
5. Run `npm start`
